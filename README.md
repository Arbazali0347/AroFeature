# 🚀 AroFeature - Feature Feedback & Voting SaaS

AroFeature is a modern SaaS platform that helps businesses collect user feedback, prioritize ideas, and build features that users actually want.

---

## ✨ Features

- 🧠 Collect user feedback بسهولة
- 👍 Voting system to prioritize ideas
- 📊 Manage feature requests in one place
- ⚡ Clean and modern UI (AI-enhanced design)
- 🔒 Scalable MERN stack architecture

---

## 🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: Tailwind CSS / Custom UI
- Animations: Framer Motion
